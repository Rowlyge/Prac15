Production Line Simulation (C, POSIX Threads)

Имитация работы производственной линии с использованием потоков и считающих семафоров POSIX.
Детали A, B и C производятся параллельно с разным временем изготовления, после чего собираются в модуль (A + B) и готовое изделие (модуль + C).

Функциональность

- Потоки производителей деталей A, B, C

- Потоки сборки модуля и конечного изделия

- Семафоры для синхронизации этапов

- Имитация задержек с помощью sleep()

- Мгновенное завершение потоков по Ctrl+C (SIGINT)

- Корректное освобождение ресурсов

Компиляция

gcc production_line.c -o production_line -pthread

Запуск

./production_line

Завершение

Нажмите Ctrl+C, чтобы остановить все потоки и корректно завершить программу.
