Производственная линия
Имитация производства изделия из деталей A(1с), B(2с), C(3с). Детали A+B собираются в модуль, затем модуль+C = изделие.

Сборка и запуск
bash
gcc production_line.c -lpthread -o production_line
./production_line
Ctrl+C - завершение работы.

Особенности
Многопоточность (pthread)

Семафоры для синхронизации

Корректная обработка завершения
